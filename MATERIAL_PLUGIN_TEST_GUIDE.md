# 素材库插件测试指南

## 修复总结

素材库插件现已完全集成到 Agent 系统中。用户可以在系统提示词中引用素材库插件，Agent 将自动识别并执行素材库检索。

## 核心特性

### 1. 系统提示词中的插件引用

在 Agent 的系统提示词中，使用以下格式引用素材库插件：

```
⊕ 自由生图-参考图
```

### 2. 自动识别与执行

当 Agent 执行时：
- 系统自动解析系统提示词中的 `⊕ 插件引用`
- 识别到 "自由生图-参考图"（包含"参考图"）
- 从 LLM 输出中智能提取关键词
- 在素材库中进行精确匹配
- 返回匹配的图片 URL

3. 在"系统提示词"部分，输入包含 `⊕ 自由生图-参考图` 的系统提示词

## 关键词提取规则

系统支持从多种格式中自动提取关键词：

### 1. 【】括号格式（优先级最高）

```
【美食摄影风】 → 提取：美食摄影风
【圣诞美食场景】 → 提取：圣诞美食场景
【蛋糕、面包、咖啡】 → 提取：蛋糕（只取第一个）
```

### 2. 引号格式

```
找"美食摄影风"的参考图 → 提取：美食摄影风
参考图库中找"真人实景" → 提取：真人实景
在图库中找"大字报" → 提取：大字报
```

### 3. 量词模式

```
一个蛋糕 → 提取：蛋糕
一只玩具 → 提取：玩具
一张贺卡 → 提取：贺卡
```

## 常见问题

### Q: 为什么找不到素材？

**A:** 检查以下几点：

1. **关键词是否正确提取？**
   - 查看控制台中的 "🔍 提取的所有关键词" 输出
   - 确认提取的关键词是否与素材库中的名称完全相同

2. **素材库中有哪些素材？**
   - 查看控制台中的 "📋 素材库中的所有名称" 输出
   - 与你想要的关键词进行比对

3. **LLM 输出是否包含搜索指令？**
   - LLM 应该在输出中明确说明要查找哪个素材
   - 例如：`在图库⊕ 自由生图-参考图 中找"美食摄影风"的参考图`

### Q: 如何添加更多素材？

**A:** 
1. 打开应用首页中的"素材库"或"资源"页面
2. 上传新的图片并设置名称
3. 确保名称与 LLM 可能会提取的关键词一致

### Q: 素材库插件可以用于其他场景吗？

**A:** 当前只支持 "自由生图-参考图"。未来可以扩展为支持多个素材库插件。

## 架构设计

### 插件识别流程

```
┌─ 系统提示词
│  └─ 包含 "⊕ 自由生图-参考图"
│
├─ 解析插件引用
│  └─ 使用正则表达式识别 ⊕ 后的插件名称
│
├─ 检查是否是素材库插件
│  └─ 检查名称是否包含 "参考图"
│
├─ 执行素材库插件
│  ├─ 从 LLM 输出中提取关键词
│  └─ 在素材库中进行精确匹配
│
└─ 返回结果
   └─ 图片 URL 或 null
```
